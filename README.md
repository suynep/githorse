# githorse
> interact w/ git as fast as a horse


## features
> we'll try to be as helpful as possible

### exploration
- akin to `git log`, essentially allowing the end-user to navigate through the commit
history
- akin to `git diff`/`git whatchanged`

#### reverting to a specific commit
- `git checkout <commit_hash>`

### tracking
- `git add <files>`

### creating commit objects
- `git commit -m <msg>`
